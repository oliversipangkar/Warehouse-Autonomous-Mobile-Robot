# UWB-based Indoor Positioning System (IPS) and Improved A* Path Planning on Mobile Robot

Hardware : DDSM115 Wheel, LPMS CU2 IMU, ULM1 UWB, Makita 18 V

IPS Features :
- DWM1000 chip
- 2D dimension positioning
- Trilateration technique (3 Anchor and 1 Tag)
- Symmetrical Double Sided Two Way Ranging (SDS-TWR)

A* Features :
- Steering cost information
- Euclidean and manhattan distance in heuristic cost
- Static obstacle
